BORDERS, MARGINS & PADDING

About

Every element that is added in HTML consists of the content itself, padding, border and margin in this order from the inside to the outside. These layers can be compared that of an onion. The content has properties of width and height that can be most commonly changed with either pixels (px) or proportion(%), while there are many different choices for units for either of these. These values can also be set to auto which will automatically pick the smallest size of the rectangle that encloses all the content. Once the size of the content has been set, padding can be adjusted. Padding is aditional space between the content and it's border. An example of this can be seen in the buttons on this page, the text is closely enclosed as the content and the space between the text and the border is the padding. Padding can be easily set with padding: 14px; as an example. This would set the space around the content on each side of it equaly to 14 pixels. for finer control over this, properties like padding-top, padding-right and so on, can be used.

Surrounding the padding of the element is the border. The border displays a rectangle around the element and it's appearance can be customised with the border-style property which can take a large variaty of different types such as dotted, solid, double and many more. If you dont want the border to show, it will have the width of 0 by default meaning that it will be invisible on the website, alternatively, border-style: none; can be used. Again, an example of this can be seen in a button object on this page. The button has a white border that is solid. borders can also be applied to one side of the element only like border-right-style and so on. The colour of the border can be set in a very similar way with the border-color property.
The most outer layer of all elements is the margin. The margin is used to add space around the border and is always transparent. Just like the other properties, the margin can be applied to either all sides at once, or to each side seperately.

This way of ogranising content is known as the "CSS Box Model" and is very important to layout of your webiste. It allows you to think about each object on the page as a rectange and makes it easier to design websites. The overall height of the full element, can be calculated as height + padding-bottom + padding-top + border-bottom-width + border-top-width + margin-bottom + margin-top.
